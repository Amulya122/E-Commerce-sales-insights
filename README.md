# E-Commerce-sales-insights
##  Overview
This project is a KPI-based E-commerce Sales Insights Dashboard developed using Python and Flask. It analyzes sales data and displays key business metrics such as total sales, total orders, and average order value through a simple web interface.

## Objectives
- Analyze e-commerce sales data
- Generate key performance indicators (KPIs)
- Visualize business insights using a dashboard
- Reduce manual data analysis effort

##  Technologies Used
- Python
- Flask
- Pandas
- HTML, CSS, JavaScript

## Features
- Total Sales Calculation
- Total Orders Count
- Average Order Value (AOV)
- API-based data fetching
- Simple and interactive dashboard
